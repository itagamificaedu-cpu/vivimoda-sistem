Antes de Começar — Checklist

Verifique cada item antes de iniciar a instalação:

  • Computador com Windows 10 ou 11 (64 bits)
  • Pelo menos 2 GB de espaço livre no disco C:
  • Conexão com a internet ativa (necessária apenas na instalação)
  • Conta de Administrador do Windows disponível
  • Antivírus desativado temporariamente (ou exceção criada)
  • Pasta do sistema copiada para o computador (ex: C:\VIVIMODA)
  • Navegador Chrome ou Edge instalado
💡
Onde colocar a pasta do sistema?
Recomendado: C:\VIVIMODA\ ou C:\Users\NomeUsuario\VIVIMODA\
Evite pastas com caracteres especiais ou acentos no caminho.

Instalação Passo a Passo

3.1 — Copiar os arquivos

  1. 1
    Copie a pasta completa do sistema para o computador da loja.
    Sugestão: C:\VIVIMODA SISTEM\confec_system\
  2. 2
    Certifique-se que todos os arquivos estão presentes:
    INSTALAR.bat, manage.py, pasta apps\, pasta templates\

3.2 — Executar o Instalador

  1. 1
    Abra a pasta do sistema no Windows Explorer
  2. 2
    Clique com o botão direito no arquivo INSTALAR.bat
  3. 3
    Selecione "Executar como Administrador"
  4. 4
    Na janela de controle de conta (UAC), clique SIM
  5. 5
    Aguarde de 3 a 10 minutos — a janela mostrará o progresso em 7 etapas
  6. 6
    Ao final, o sistema pergunta se deseja abrir agora. Digite S e pressione Enter.
⚠️
Se aparecer "Windows protegeu seu computador":
Clique em "Mais informações" e depois em "Executar assim mesmo". Isso acontece porque o arquivo não tem certificado digital.

Início Automático com o Windows

O instalador configura automaticamente o servidor para iniciar junto com o Windows via Agendador de Tarefas. Veja como verificar:

5.1 — Verificar se está configurado

schtasks /Query /TN "VIVIMODA Sistema"

Execute no Prompt de Comando. Deve mostrar "VIVIMODA Sistema" com status Pronto.

5.2 — Configurar manualmente (se não foi automático)

  1. 1
    Pressione Win + R, digite taskschd.msc, Enter
  2. 2
    Clique em "Criar Tarefa Básica" no painel direito
  3. 3
    Nome: VIVIMODA Sistema → Próximo
  4. 4
    Gatilho: "Ao fazer logon" → Próximo
  5. 5
    Ação: "Iniciar um programa" → Próximo
  6. 6
    Programa: wscript.exe
    Argumentos: /nologo "C:\VIVIMODA SISTEM\confec_system\iniciar_servidor.vbs"

5.3 — Tempo de espera

⏱️
Após ligar o computador, aguarde 5 a 10 segundos antes de abrir o navegador. O servidor precisa desse tempo para inicializar.

Acesso em Rede (múltiplos computadores)

O sistema roda em um computador principal (servidor) e pode ser acessado de outros computadores na mesma rede local (Wi-Fi ou cabo).

6.1 — Descobrir o IP do computador servidor

  1. 1
    No computador onde o sistema está instalado, pressione Win + R
  2. 2
    Digite cmd e pressione Enter
  3. 3
    Digite o comando abaixo e pressione Enter:
ipconfig

Procure por "Endereço IPv4", exemplo: 192.168.1.105

6.2 — Acessar de outro computador

Em qualquer outro computador da mesma rede, abra o navegador e acesse:

http://192.168.1.105:8000 (substitua pelo IP real)

6.3 — Liberar o Firewall do Windows

Se outros computadores não conseguirem acessar, libere a porta 8000:

  1. 1
    Abra o Painel de Controle → Firewall do Windows Defender
  2. 2
    Clique em "Configurações Avançadas"
  3. 3
    Clique em "Regras de Entrada" → "Nova Regra"
  4. 4
    Tipo: Porta → TCP → Porta: 8000 → Permitir → Todos os perfis → Nome: VIVIMODA

Ou execute este comando no Prompt como Administrador:

netsh advfirewall firewall add rule name="VIVIMODA" dir=in action=allow protocol=TCP localport=8000
⚠️
Atenção de segurança: O sistema foi projetado para rede local (LAN). Não exponha a porta 8000 para a internet sem configurar HTTPS e autenticação adicional.

Configurar Backup Automático

7.1 — Backup manual (já incluso)

O arquivo BACKUP_DADOS.bat já está incluso. Execute-o para criar um backup:

  • Cria uma pasta na Área de Trabalho com a data de hoje
  • Copia o banco de dados e as mídias (fotos de produtos)
  • Abre a pasta automaticamente ao final

7.2 — Backup automático semanal (Agendador de Tarefas)

  1. 1
    Pressione Win + R, digite taskschd.msc
  2. 2
    Clique em "Criar Tarefa Básica"
  3. 3
    Nome: VIVIMODA Backup Semanal
  4. 4
    Gatilho: Semanalmente → Sábado → 18:00
  5. 5
    Programa: C:\VIVIMODA SISTEM\confec_system\BACKUP_DADOS.bat

7.3 — O que fazer com o backup

✅
Copie a pasta de backup para um pen drive ou Google Drive. Nunca deixe o único backup no mesmo computador do sistema — se o HD pifar, você perde tudo.

7.4 — Restaurar um backup

  1. 1
    Execute PARAR_SERVIDOR.bat para parar o sistema
  2. 2
    Execute RESTAURAR_BACKUP.bat
  3. 3
    Arraste o arquivo banco_de_dados.sqlite3 do backup para a janela aberta
  4. 4
    Pressione Enter e aguarde a confirmação
  5. 5
    Abra o sistema normalmente pelo atalho da Área de Trabalho
8
Desinstalar o Sistema
  1. 1
    Execute PARAR_SERVIDOR.bat para encerrar o servidor
  2. 2
    Remova a tarefa do Agendador de Tarefas:
    schtasks /Delete /TN "VIVIMODA Sistema" /F
  3. 3
    Exclua o atalho da Área de Trabalho (VIVIMODA Sistema.lnk)
  4. 4
    Faça backup dos dados antes de excluir a pasta!
    Execute BACKUP_DADOS.bat e guarde em local seguro
  5. 5
    Delete a pasta completa do sistema
⚠️
O Python e os pacotes instalados não são removidos automaticamente. Para remover o Python, use o Painel de Controle → Programas.

Resolução de Problemas
❌ "Python não foi encontrado"

Causa: Python não está instalado ou não está no PATH do Windows.

Solução:
  1. Baixe Python em python.org/downloads (versão 3.11 recomendada)
  2. Na instalação, marque "Add Python to PATH"
  3. Reinicie o computador
  4. Execute INSTALAR.bat novamente
❌ O sistema não abre no navegador
Tente estes passos em ordem:
  1. Abra o Chrome e acesse manualmente: http://localhost:8000
  2. Se não carregar, execute PARAR_SERVIDOR.bat e depois clique no atalho novamente
  3. Verifique se o antivírus está bloqueando o Python. Adicione uma exceção para a pasta do sistema
  4. Execute INSTALAR.bat novamente como Administrador
❌ Antivírus bloqueando / "ameaça detectada"
Causa: Antivírus pode bloquear o Python ou o arquivo .bat/vbs.
Solução:
  1. Abra o antivírus e vá em Exclusões / Exceções
  2. Adicione a pasta completa do sistema como exceção
  3. Adicione também o executável do Python (caminho em python_path.txt)
  4. Execute o instalador novamente
❌ "Porta 8000 já está em uso"
Causa: Outra instância do servidor está rodando.
Solução: Execute PARAR_SERVIDOR.bat e aguarde 5 segundos antes de abrir novamente.
❌ Tela de login aparece mas dá "Senha inválida"

Usuário: admin    Senha: confec@2024

Se não funcionar, redefina pelo terminal:
python manage.py changepassword admin --settings=confec_system.settings.local_sqlite
❌ Instalação parou na etapa 2 (pacotes)
Causa: Sem conexão com internet ou firewall corporativo.
Solução:
  1. Verifique a conexão com a internet
  2. Tente desativar temporariamente o firewall
  3. Execute o instalador novamente — ele continua de onde parou
❌ Atalho da Área de Trabalho não funciona
Solução: Verifique se o arquivo VIVIMODA.vbs está na pasta do sistema. Se não estiver, execute INSTALAR.bat novamente para recriá-lo.
